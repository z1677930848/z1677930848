# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.15] - 2024-XX-XX

### Added
- User portal integration
- Dual port support (admin/user)
- One-click upgrade feature
- SSL certificate management improvements

### Changed
- Migrated from Tailwind CSS to TeaUI framework
- Optimized admin login flow
- Improved UI consistency

### Fixed
- User 404 error issues
- Login authentication problems
- Icon rendering issues
- Boards page diagnosis

## [1.0.11] - Previous Release

### Initial Features
- Multi-user support
- Log audit
- Cluster management
- HTTP/HTTPS/TCP/UDP protocol support
- WAF (Web Application Firewall)
- Cache management
- DNS auto-resolution
- SSL certificate management
- IP whitelist/blacklist
- Access logs and statistics
